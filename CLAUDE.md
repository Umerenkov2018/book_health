Задание: Создай современное одностраничное веб-приложение (HTML/JS/Tailwind) "Дневник здоровья и активности".

Дизайн: Минималистичный, темная тема (dark mode), адаптивный интерфейс. Использовать карточную структуру. Адаптировать под интерфейс iPhone 17 pro.

Функционал:

Блок ежедневных факторов: >    - Трекеры с быстрыми кнопками: Сон (часы), Кофе (чашки), Вода (литры), Уровень стресса (1-10).

Выпадающий список погоды (Ясно, Облачно, Осадки, Перепад давления).

Блок фиксации приступов (События):

Кнопка "Добавить приступ мигрени" и "Добавить приступ (простата)".

При клике открывается форма: Интенсивность (1-10), Локализация, Принятые лекарства/действия, Комментарий.

Визуализация: >    - Календарь-сетка активности (как на GitHub), где клетки подсвечиваются, если в этот день была зафиксирована боль или заполнена активность.

Хранение данных:

Автоматическое сохранение в localStorage.

Кнопки "Экспорт в JSON" (скачать файл) и "Импорт из JSON" (загрузить файл).

Таблица истории: Список последних записей с возможностью удаления.

Технические требования: Весь код должен быть в одном файле. Использовать чистый JavaScript без внешних тяжелых библиотек (кроме Tailwind через CDN).
